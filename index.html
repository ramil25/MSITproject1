Project 1: Experiment on Web Browser speed


Abstract

On this experement the researcher choose four web browsers and test it one by one in terms of startup speed, loading speed of css and javascript and also the used memory of the applications when its running. The four web browser that chose by researcher are the Chrome, Safari, Opera and Brave. After the procedures of testing in that four web browsers the researcher found that every web browser is better than others in term of other aspect.
